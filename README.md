# 🌪️ Underground Rap Database (Flask Web App)

მოგესალმებით **Underground Rap Database**-ში! ეს არის დინამიური ვებ-აპლიკაცია, რომელიც შექმნილია Flask ფრეიმვორქის ბაზაზე. პლატფორმა წარმოადგენს თანამედროვე "New Wave" და "Underground" ჰიპ-ჰოპ არტისტების (2slimey, xaviersobased, che და ა.შ.) კატალოგს, სადაც მომხმარებლებს შეუძლიათ თვალი ადევნონ მათ შემოქმედებას, ხოლო ავტორიზებულ იუზერებს — დატოვონ "ქუჩის რეცენზიები" (Street Reviews).

---

## 🚀 ფუნქციონალი (Features)

- **მთლიანი კატალოგი (CRUD):** მთავარ გვერდზე გამოტანილია ყველა აქტიური არტისტი თავისი დებიუტის წლითა და ალბომის ქავერით.
- **მომხმარებელთა ავტორიზაცია:** სრულფასოვანი რეგისტრაციისა და ავტორიზაციის (Login/Register) სისტემა მართული `Flask-Login`-ით და პაროლების უსაფრთხო დაჰეშირებით (`Werkzeug`).
- **ქუჩის რეცენზიები (Street Reviews):** ნებისმიერ ავტორიზებულ მომხმარებელს შეუძლია დატოვოს კომენტარი/რეცენზია კონკრეტული არტისტის გვერდზე, სადაც ავტორის სახელი ავტომატურად გამოჩნდება მარჯვენა მხარეს.
- **ადმინ პანელი / კონტროლი:** ადმინისტრატორს (`admin` იუზერნეიმით) აქვს ექსკლუზიური უფლება, დაამატოს ახალი არტისტები, ჩაასწოროს არსებული მონაცემები ან წაშალოს ისინი პირდაპირ ინტერფეისიდან.
- **მონაცემთა ავტომატური ინიციალიზაცია:** `init_db.py` სკრიპტი, რომელიც ავტომატურად ასუფთავებს ბაზას და საწყის ეტაპზევე ყრის 5 წამყვან Underground არტისტს.

---

## 🛠️ ტექნოლოგიური სტეკი (Tech Stack)

- **Backend:** Python, Flask
- **Database / ORM:** SQLite, Flask-SQLAlchemy (Relational Database: Users ↔️ Reviews ↔️ Artists)
- **Security:** Werkzeug (Password Hashing)
- **Authentication:** Flask-Login
- **Frontend:** HTML5, Jinja2 Templates, CSS3, Bootstrap 5 (Dark theme styled)
- **Forms:** Flask-WTF, WTForms (Validation)
